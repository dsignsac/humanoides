# Skill: GSAP Router
# Path: .agent/skills/gsap/router.md

## Descripción
Gestión de transiciones de página fluidas y orquestación de entrada/salida de componentes en cambios de ruta.

## Capacidades
- Animaciones "Exit" coordinadas antes del cambio de DOM.
- Transiciones "Page Overlay" y persistencia de estados.
- Integración con React Router o Next.js.

## Flujo Operacional
1. Interceptar el cambio de ruta.
2. Ejecutar timeline de salida.
3. Resolver la ruta y ejecutar timeline de entrada.
