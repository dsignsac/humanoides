# Skill: GSAP Awwwards Template
# Path: .agent/skills/gsap/awwwards.md

## Descripción
Arquitectura de referencia para proyectos de alto impacto visual. Implementa el "Golden Stack" para sitios de nivel Awwwards.

## Stack Técnico
- **Framework**: React 19 + Vite.
- **Motion**: GSAP + ScrollTrigger + Custom Cursors.
- **CSS**: Tailwind CSS 4.
- **Performance**: 60fps targets, optimización de assets.

## Protocolo de Despliegue (Vercel)
- `vercel build --prod`
- `vercel deploy --prebuilt --prod`
